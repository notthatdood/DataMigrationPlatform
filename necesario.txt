dependencias que instalé

pip install pika--upgrade
pip install elasticsearch
pip install mysql
pip install mariadb
pip install prometheus-client

1. Primero navegamos en powershell a donde esté el repositorio

2 Correr el script: createEverything.ps1 (dura como 10 mins)

3 Luego hacer los port forwards de mariadb, elastic y rabbitmq
Cada uno en una ventana de powershell distinta
    kubectl port-forward --namespace default svc/components-rabbitmq 5672:5672
    kubectl port-forward --namespace default svc/databases-mariadb 3306:3306
    kubectl port-forward --namespace default svc/databases-elasticsearch 9200:9200

4 Correr el script: populateMariaDB.py (está dentro de scripts/pythonScripts/fillDBs)

5 Correr el script: insertJob.py (está dentro de scripts/pythonScripts/fillDBs)

6 Abrimos un navegador y vamos a localhost:8000 para ver las métricas de prometheus

7 Luego vamos a correr los scripts que deberían estar en los pods de python
Cada uno en una ventana de powershell distinta
Estos se encuentran dentro de la carpeta "DataMigrationPlatform\scripts\pythonScripts\deploymentLogic"
Se deben correr en el siguiente orden: 
    1. orchestratorLogic.py
    2. MySQLConnectorLogic.py
    3. SQLProcessorLogic.py
    4. RegexProcessor.py
    5. ElasticsearchPublisherLogic.py

8 Para ver el resultado en el índice persona corra el siguiente script (si le cambia el nombre al índice se debe cambiar en el script)
    (Se encuentra enla carpeta: DataMigrationPlatform\scripts\pythonScripts\)
    viewElastic.py
